# shah18122.github.io

A live **bookshelf** of my public GitHub projects, served at
https://shah18122.github.io — it fetches my repos from the GitHub API on
every load, so it always reflects my current public repositories (deleted
repos disappear automatically). Includes a Home button to
https://www.thesmitshah.com.

Single self-contained `index.html` — no build, no dependencies.
